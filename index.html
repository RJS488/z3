            <p>The Z3 was the world's first functional automatic digital computer. Developed by German engineer Konrad Zuse in 1941, it was one of the first machines capable of performing calculations based on programs.</p>

            <h2>Features of the Z3</h2>

            <ul>

                <li>Used relays instead of electronic components</li>

                <li>Had a 22-bit word length</li>

                <li>Capable of performing arithmetic and logical operations</li>

                <li>Programmed using punched tape</li>

            </ul>



            <h2>A Milestone in Computing History</h2>

            <p>Although the Z3 was never commercially used, it is considered a crucial step in the development of modern computing technology. The Z3 paved the way for later computers like the ENIAC and today's systems, which perform millions of calculations per second.</p>

                    } else {

                        result = "Error: Invalid input!";

                    }

                }

                    let numbers = input.match(/\d+/g);

                        result = parseInt(numbers[1]) - parseInt(numbers[0]);

                    } else {
